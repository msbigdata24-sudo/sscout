Как пользоваться Сигнал-Скаутом

Шаг 1 — Бриф

  1. Укажите ваше имя — попадёт в историю.
  2. Введите сайт клиента и нажмите «Заполнить по сайту» — подставятся ниша и черновик запросов.
  3. Выберите регионы по федеральным округам (ЦФО, СФО и т.д.) — как клиент планирует ГЕО.
  4. Проверьте поисковые запросы — только коммерческие («изготовление…», «аренда…»), без бренда клиента.
  5. Нажмите «Сохранить бриф» перед запуском.
  6. Нужна копия на компьютер — «Скачать бриф» (файл JSON). Потом можно вернуть через «Загрузить из файла».

Шаг 2 — Запуск

  1. Убедитесь, что в брифе указаны ID и ключ XMLRiver (или они заданы на сервере).
  2. Нажмите «Запустить сбор» — смотрите лог в реальном времени.
  3. Если остановили — «Продолжить сбор» с того же места.

Шаг 3 — Результаты

  • Таблица: сайт конкурента, компания, регион, телефоны, источник.
  • Фильтр по региону и поиск по таблице — сверху.
  • Экспорт в Excel — кнопка в разделе «Результаты».
  • Статус «агрегатор» / «исключён» — это не конкурент, а каталог или мусор.

Куда смотреть при проблемах

  • Лог в реальном времени — на каком шаге упало (разбор сайта, Яндекс, обход).
  • История запросов — прошлые прогоны, кто запускал и с какими запросами.
  • Ошибка SSL на старом сайте клиента — попробуйте http:// вместо https://.
  • «Нет сайтов после фильтров» — расширьте запросы или регионы, проверьте баланс XMLRiver.

Нужно делать

  • Сохранять бриф перед каждым запуском.
  • Указывать своё имя в поле «Кто запускает».
  • Проверять запросы вручную после автозаполнения.
  • Исключать сайт клиента и агрегаторы (Авито, 2ГИС, Пульсцен).
  • Использовать телефоны только для рабочих задач Стратеикс.

Нельзя

  • Добавлять в запросы бренд клиента — в выдаче будет сам клиент, не конкуренты.
  • Использовать информационные запросы («что такое», «ГОСТ», «рейтинг») — попадут статьи, не заводы.
  • Запускать десятки сборов подряд без паузы — тратится XMLRiver и нагрузка на сервер.
  • Передавать выгрузку третьим лицам без согласования.
  • Спамить найденные номера — только целевой B2B-контакт по регламенту Стратеикс.
